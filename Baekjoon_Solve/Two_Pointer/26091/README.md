### 어려웠던 점 / 문제의 핵심

[고양이 카페(28353번)](https://www.acmicpc.net/problem/28353)과 동일한 문제이다. 고양이 카페 문제의 경우 상한값이기 때문에 조건을 만족하지 않을 때  `e`를 왼쪽으로 밀어주는 것이었고, 이 문제는 하한값이기 때문에 두 요소의 값이 K 미만이면 `s`를 오른쪽으로 밀어서 값을 증가시켜줘야 한다.
