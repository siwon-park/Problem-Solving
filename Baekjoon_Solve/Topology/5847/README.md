## [골3] Milk Scheduling (5847)

[https://www.acmicpc.net/problem/5847]()

### 문제 유형

그래프 이론, 위상 정렬, 다이나믹 프로그래밍

<br>

### 어려웠던 점 / 문제의 핵심

위상 정렬을 했을 때, 모든 작업을 다 할 때까지 걸리는 최소 시간을 구하는 문제이다.

위상 정렬 + 다이나믹 프로그래밍 기법을 활용하면 되며, 다음 작업을 하기 위해서는 이전 작업이 다 끝나야 하기 때문에 이전 작업들 중 가장 오래 걸린 시간을 선택하여 현재 작업 시간에 더해 누적하는 방향으로 다이나믹 프로그래밍 배열을 구성하면 된다.

`dp[nxt] = Math.max(dp[nxt], dp[cur.node] + time[nxt])`

- dp[cur] = cur 단계까지 작업을 수행하는데 걸린 최대 시간
