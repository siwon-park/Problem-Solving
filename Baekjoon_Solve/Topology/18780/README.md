## [골3] Timeline(18780번)

[https://www.acmicpc.net/problem/18780]()

### 문제 유형

DP, 그래프 이론, 위상 정렬

<br>

### 어려웠던 점 / 문제의 핵심

위상 정렬을 통해 각 작업(세션)을 수행할 수 있는 최소 일자를 찾는 문제이다.

방향 비순환 그래프에서 각 작업 간 선후행 관계에 따른, 작업의 최소 수행 일자는 그 작업을 수행할 수 있는 최대 일자를 의미한다. 즉, 이전 작업들이 모두 끝나야 다음 작업을 할 수 있기 때문에 이전 작업의 최대 시간이 후행 작업을 수행할 수 있는 최소 시작 시간이 된다.

dp 배열에 이를 기록해두면서 위상 정렬을 수행하면 각 작업의 최소 수행 가능 일자를 찾을 수 있다.
