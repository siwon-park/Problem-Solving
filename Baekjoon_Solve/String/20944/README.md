## [브3] 팰린드롬 척화비 (20944번)

https://www.acmicpc.net/problem/20944

### 문제 유형

문자열

<br>

### 어려웠던 점 / 문제의 핵심

없다.

수미 상관은 앞 뒤가 같고, 팰린드롬은 뒤집었을 때가 같은 문자열이다.

수미 상관이면서 팰린드롬인 문자열을 출력하려면, 그냥 문자열 `"a"`에 주어지는 문자열의 길이인 `N`에 맞게 출력만 하면 된다.

```
"a" * (N // 2) + "a" * (N - 2 * (N // 2)) + "a" * (N // 2)
```
