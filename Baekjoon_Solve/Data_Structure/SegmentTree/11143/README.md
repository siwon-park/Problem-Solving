## [골1] Beads (11143번)

https://www.acmicpc.net/problem/11143

### 문제 유형

자료 구조, 세그먼트 트리

<br>

### 어려웠던 점 / 문제의 핵심

구간 합을 구하는 세그먼트 트리를 구성하여 쿼리에 대한 처리를 하면 된다.

- `P i a`: i번 상자에 구슬을 a만큼 추가
- `Q i j`: 구간 [i, j]에 존재하는 구슬의 합 출력

 시간이 조금 더 걸리더라도 메모리를 최대한 덜 쓰는 방향으로 구현해보았다.

<br>
