## [골1] Hotel Rooms (32322번)

[https://www.acmicpc.net/problem/32322]()

### 문제 유형

자료 구조, 세그먼트 트리

<br>

### 어려웠던 점 / 문제의 핵심

구간 합을 구하는 세그먼트 트리이다.

A 연산일 때는 주어진 범위 l, r에 대해서 예약 가능한 방의 개수 합을 구하면 되므로 세그먼트 트리 구간 [l, r]의 합을 구하면 된다.

R 연산일 때는 주어진 idx값 위치의 방을 예약하면 되므로 구간 [idx, idx]의 세그먼트 트리 값을 -1하면 된다. (단, R연산이 들어왔을 때 idx 방의 위치는 항상 예약 가능하다고 가정한다.)
