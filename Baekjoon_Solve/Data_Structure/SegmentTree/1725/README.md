## [플5] 히스토그램 (1725번)

https://www.acmicpc.net/problem/1725

### 문제 유형

자료 구조, 세그먼트 트리, 분할 정복, 스택

<br>

### 어려웠던 점 / 문제의 핵심

구간의 최솟값의 인덱스를 저장하고 있는 세그먼트 트리를 구성하고,

구간 [1, N]에서 시작해서 해당 구간의 최솟값이 위치한 인덱스를 찾고 구간의 직사각형 너비를 구한 다음에 인덱스를 기준으로 구간을 분할 정복하여 너비의 최댓값을 구하면 된다.
