## [골4] PPAP (16120번)

https://www.acmicpc.net/problem/16120

### 문제 유형

문자열, 스택, 그리디, 자료구조

<br>

### 어려웠던 점 / 문제의 핵심

PPAP 문자열이란 문자열 P에서 "시작"한다.

입력 받은 문자열을 스택으로 구성하여 스택의 상위 4개의 연속된 문자열이면 "P"로 교체한다. 해당 행동은 PPAP 문자열 중 P를 PPAP로 바꾼 행동의 역이다.

위의 행동을 반복했을 때, 스택에 남아있는 문자열이 P 하나 뿐이라면 PPAP 문자열이다.
