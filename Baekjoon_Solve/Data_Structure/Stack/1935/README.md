### 어려웠던 점 / 문제의 핵심

스택의 원리를 이용해서, 연산자가 들어왔을 때, 스택의 요소를 2개 뽑아서 연산 결과를 다시 스택에 집어 넣는 과정을 반복하면 된다.

각 문자에 해당하는 숫자는 Map 자료형을 쓰면 된다.

#### int → char 형 변환

```java
char s = (char) (65 + '0');
char s = (char) (n + i);
```

#### 소수점 출력
