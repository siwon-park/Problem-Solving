## [실2] N번째 큰 수 (2075번)

https://www.acmicpc.net/problem/2075

### 문제 유형

자료 구조, 우선 순위 큐

<br>

### 어려웠던 점 / 문제의 핵심

최소 우선 순위 큐를 구성하여, 우선 순위 큐의 크기가 N + 1이 되었을 때, poll 연산을 통해 요소를 빼는 과정을 반복하여 우선 순위 큐의 크기가 N을 유지하도록 한다.

최종적으로 크기가 N인 최소 우선 순위 큐에서 루트에 있는 요소가 N번째 큰 수이다.
