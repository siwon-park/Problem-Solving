### 문제 유형

### 어려웠던 점 / 문제의 핵심

우선순위 큐의 크기가 2이상인 동안 반복하여 우선순위 큐 제일 위에 있는 상위 2개의 파일을 합치고, 합친 파일의 크기를 정답에 누적한다. 그 후 합친 파일을 다시 우선순위 큐에 삽입한다.
