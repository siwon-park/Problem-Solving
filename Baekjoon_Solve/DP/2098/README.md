https://www.acmicpc.net/problem/2098

### 문제 유형

DP, 비트마스킹

<br>

### 어려웠던 점 / 문제의 핵심

비트 마스킹을 활용한 메모이제이션으로 풀면 된다.

단, `W[i][i] = 0 `인 것은 0의 비용으로 갈 수 있다는 의미이지만, `W[i][j] = 0`은 갈 수 없다는 의미이다. 이를 구분해서 문제를 풀어야 한다.
