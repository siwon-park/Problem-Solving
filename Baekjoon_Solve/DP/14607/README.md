## [실3] 피자 (Large) (14607번)

https://www.acmicpc.net/problem/14607

### 문제 유형

수학, DP

<br>

### 어려웠던 점 / 문제의 핵심

짝수냐, 홀수냐에 따라 나눠서 풀면 DP로 쉽게 해결할 수 있다.

공간 복잡도는 O(logN)이기 때문에 N이 10억이라도 상관없다.

DP말고, 수학적 귀납법으로 정답이 N * (N - 1) / 2임을 증명해서 풀어도 된다.
