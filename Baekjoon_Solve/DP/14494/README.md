## [실3] 다이나믹이 뭐예요? (14494번)

https://www.acmicpc.net/problem/14494

### 문제 유형

DP

<br>

### 어려웠던 점 / 문제의 핵심

`dp[0][0] = 1`로 두고 DP를 하면 된다.

점화식은 `dp[i][j] = (dp[i - 1][j] + dp[i][j - 1] + dp[i - 1][j - 1]) % MOD`이다.

단, 연산 도중에 오버 플로우가 날 수도 있기 때문에, 나눠서 계산하는 것이 좋다.
