### 어려웠던 점 / 문제의 핵심

방문할 수 있는 쉼터의 개수를 메모이제이션을 통해 기록하면 된다.

다음에 방문할 노드가 현재 높이보다 높을 경우에만 방문할 수 있는 노드의 수를 갱신한다.

```java
if (height[nxt] > height[cur]) v = Math.max(v, 1 + dfs(nxt));
```
