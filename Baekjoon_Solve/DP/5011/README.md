## [골5] 문제

[https://www.acmicpc.net/problem/5011]()

### 문제 유형

DP, BFS

<br>

### 어려웠던 점 / 문제의 핵심

DP와 BFS를 사용해야 풀 수 있는 문제이다.

오른쪽 아래로만 이동했을 때의 총 경로의 수는 DP를 통해 구하면 된다.

- 점화식: `dp[i][j] = dp[i - 1][j] + dp[i][j - 1]`

다만, 경로의 수를 2^31 - 1로 나눈 나머지를 구해야 하기 때문에 오른쪽 아래로만 이동했을 때 결과값이 0인 경우도 존재한다.

결과값이 0일 때 우하향해서 도달할 수 없다고 해버리면 올바른 정답이 아니기 때문에, 우하향만 이동하는 BFS와 4방향 전부 이동하는 BFS를 통해서 판별하면 된다.

- 우하향만 이동하는 BFS가 가능할 경우: dp의 결과값 출력
  - 따라서 우하향 이동이 가능할 때만 dp를 돌려주면 된다.
- 우하향은 불가능한데, 4방향은 가능할 경우: "THE GAME IS A LIE" 출력
- 4방향도 불가능할 경우: "INCONCEIVABLE"
