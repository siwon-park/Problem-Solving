### 어려웠던 점 / 문제의 핵심

피보나치 수열을 DP를 통해 구한다.

N개의 타일로 구한 직사각형의 둘레는 N번째 피보나치 수 x 2와 N + 1번째 피보나치 수 x 2의 합이다.
