## [골4] 행성 연결 (16398번)

https://www.acmicpc.net/problem/16398

### 문제 유형

그래프 이론, 최소 스패닝 트리(MST)

<br>

### 어려웠던 점 / 문제의 핵심

모든 행성을 연결하면서 비용을 최소로 하고자 하므로 최소 스패닝 트리를 구성하면 된다.

최소 스패닝 트리의 연결 간선 수는 노드 수(N) - 1이므로, 조금 더 빠른 풀이를 위해 간선 수가 N - 1개가 되었을 때 break하면 된다.
