## [골4] Networking (3803번)

https://www.acmicpc.net/problem/3803

### 문제 유형

그래프 이론, 최소 스패닝 트리

<br>

### 어려웠던 점 / 문제의 핵심

기본적인 최소 스패닝 트리 문제이다. 

각 테스트 케이스별로 최소 스패닝 트리를 구축하기 위한 최소 비용을 출력하면 된다.

비용이 다른 간선 정보가 중복으로 주어질 수 있기 때문에 모든 간선 정보를 받아 비용이 작은 순으로 오름차순 정렬해서 문제를 풀어야 한다.
