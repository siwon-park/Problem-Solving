https://www.acmicpc.net/problem/15835

### 문제 유형

자료 구조, 분리 집합, MST

<br>

### 어려웠던 점 / 문제의 핵심

문제를 해석하면 각 케이스별로 MST를 구성했을 때 드는 최소 비용을 구해야 하는 문제임을 알 수 있다.

모든 체크 포인트를 방문하면서 최단 거리로 이동해야 한다고 했기 때문에 그렇다.

크게 고려할 부분이 없는 통상적인 MST문제이다.
