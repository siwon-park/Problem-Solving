### 어려웠던 점 / 문제의 핵심

비기게 되는 시점은 중요하지 않다.

`마지막 득점 시간을 계속 기록`하면서 팀이 이기고 있을 때마다, `현재 득점 시간 - 마지막 득점 시간을 누적`한다.

마지막으로 이기고 있는 팀에 대해서 `48분 - 마지막 득점 시간`을 누적 계산해주면 된다.

너무 어렵게 생각해서 여러 경우의 수를 나누다 보니 어떻게 풀어야 할지 감이 오지 않았다.
