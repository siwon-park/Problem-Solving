## [골2] 물통 (14867번)

https://www.acmicpc.net/problem/14867

### 문제 유형

자료 구조, 그래프 이론, 그래프 탐색, BFS

<br>

### 어려웠던 점 / 문제의 핵심

10만 x 10만의 방문 배열을 선언하면 메모리가 터질게 분명하기 때문에 set배열을 사용하였다.

어차피 생각보다 경우의 수는 그렇게 많이 나오지 않는다.

나올 수 있는 경우의 수는 a x b보다 훨씬 작다.

- 한쪽이 꽉 차있는 경우 `(a, 0 ~ b)`, `(0 ~ a, b)`
- 한쪽이 비어있는 경우 `(0, 0 ~ b)`, `(0 ~ a, 0)`
