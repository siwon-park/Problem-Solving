## [실1] 신아를 만나러

https://www.acmicpc.net/problem/6146

### 문제 유형

그래프 이론, 그래프 탐색, 너비 우선 탐색

<br>

### 어려웠던 점 / 문제의 핵심

좌표의 범위가 -500 <= y, x <= 500이므로 모든 좌표에 500을 더해서 양수로 만든 다음 2차원의 배열을 활용하여 BFS 탐색을 하면 된다.
