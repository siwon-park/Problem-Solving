## [골3] 모양 만들기 (16932번)

https://www.acmicpc.net/problem/16932

### 문제 유형

그래프 이론, 그래프 탐색, BFS, DFS

<br>

### 어려웠던 점 / 문제의 핵심

너비 우선 탐색(BFS)를 통해 각 좌표별 연결 컴포넌트 번호와 연결 컴포넌트별 크기를 구한다.

1개의 0인 영역만 1로 변경한다고 했으므로 변경된 영역의 크기는 1부터 시작하고, 상하좌우로 위치한 연결 컴포넌트별 크기를 중복 없이 합산하여 최댓값을 구하면 된다.
