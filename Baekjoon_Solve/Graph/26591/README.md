https://www.acmicpc.net/problem/26591

### 문제 유형

그래프 이론, 그래프 탐색, 0-1 BFS

<br>

### 어려웠던 점 / 문제의 핵심

그래프의 크기도 8x8이라 시간 복잡도를 크게 염려하지 않아도 된다.
