## [골5] Coast Length (11183번)

https://www.acmicpc.net/problem/11183

### 문제 유형

그래프 이론, 그래프 탐색, BFS, DFS, 플러드 필

<br>

### 어려웠던 점 / 문제의 핵심

섬의 해안의 총 길이를 구하는 문제이다.

단, 섬 안에 있는 물(섬으로 둘러싸인 물; 호수) 혹은 호수 안에 있는 섬은 해안으로 간주하지 않는다.

맵의 크기를 상하좌우 한 칸씩 늘리고, (0, 0)을 바다로 간주하여 바다인 영역을 표시한다. 

사전에 맵을 만들 때 값이 1인 좌표를 미리 큐에 넣어둔다. 해당 큐에서 요소를 뽑으면서 상하좌우 주변을 탐색했을 때, 맵의 값이 0이면서 방문 표시되어 있는 곳(=바다인 부분)일 경우 해안으로 간주하여 +1씩 증가시키면 답을 찾을 수 있다.
