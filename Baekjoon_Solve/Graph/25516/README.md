## [실2] 거리가 K이하인 트리 노드에서 사과 수확하기 (25516번)

[https://www.acmicpc.net/problem/25516]()

### 문제 유형

그래프 이론, 그래프 탐색, BFS, DFS

<br>

### 어려웠던 점 / 문제의 핵심

트리를 BFS나 DFS로 탐색하면서 루트 노드에서 출발했을 때 도달 가능한 최단 거리가 K이하인 노드들에 대해서 사과의 개수를 구하면 된다.

굳이 부모-자식을 양방향으로 이어주지 않아도 된다.
