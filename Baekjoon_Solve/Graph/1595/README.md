https://www.acmicpc.net/problem/1595

1번 노드에서 가장 먼 거리에 있는 노드를 구한 다음에 해당 노드에서 다시 가장 먼 거리에 있는 노드를 구하면 된다.

트리의 가장 긴 지름을 구하는 문제이다.
