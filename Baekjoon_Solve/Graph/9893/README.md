## [골5] Paths (9893번)

https://www.acmicpc.net/problem/9893

### 문제 유형

그래프 이론, 그래프 탐색, 다익스트라

<br>

### 어려웠던 점 / 문제의 핵심

0 → 1로 가는 경로 중 최단 링크 경로 중 최소 비용을 찾는 문제이다.

2차원의 다익스트라 배열을 통해 `distance[i][j]`를 다음과 같이 정의한다.

- 0부터 i까지 j번 링크로 갔을 때의 최소 비용.
