## [골3] 점령 (32620번)

[https://www.acmicpc.net/problem/32620]()

### 문제 유형

자료 구조, 그래프 이론, 그래프 탐색, BFS, 우선순위 큐

<br>

### 어려웠던 점 / 문제의 핵심

우선순위 큐를 활용한 BFS 탐색을 하면 된다.

점령에 필요한 기력이 가장 작은 수치를 더 높은 우선순위로 가지는 큐를 구성하여, 큐 맨 앞에 나온 점령 필요치보다 현재 총 점령치가 크거나 같다면 해당 노드를 점령한다.
