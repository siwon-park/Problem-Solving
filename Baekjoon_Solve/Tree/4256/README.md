## [골2] 트리 (4256번)

[https://www.acmicpc.net/problem/4256]()

### 문제 유형

트리, 분할 정복, 재귀

<br>

### 어려웠던 점 / 문제의 핵심

트리의 전위 순회, 중위 순회를 알면 후위 순회를 알아 낼 수 있음을 이용하는 문제이다.

전위 순회, 중위 순회의 루트 위치와 왼쪽, 오른쪽 서브 트리의 인덱스를 활용해 재귀 호출, 분할 정복하여 후위 순회를 알아내면 된다.
