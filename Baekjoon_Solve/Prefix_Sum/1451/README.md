## [골4] 직사각형으로 나누기 (1451번)

https://www.acmicpc.net/problem/1451

### 문제 유형

누적 합, 브루트포스

<br>

### 어려웠던 점 / 문제의 핵심

2차원의 배열 누적 합을 계산한 다음에 `ㅗ`, `ㅜ`, `||`, `=`, `ㅓ`, `ㅏ` 이 6가지 모양으로 삼분할 했을 때, 직사각형의 합들의 곱의 최댓값을 구하면 된다.

단, 행이 1일 경우엔 `||`, 열이 1일 경우엔 `=` 모양에 대해서만 계산해야 한다.
