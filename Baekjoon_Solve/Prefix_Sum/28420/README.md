### 어려웠던 점 / 문제의 핵심

2차원의 배열 누적 합을 계산하는 문제이다.

주어진 가능한 3개의 모양에 대해 올바른 2차원의 배열 누적 합을 계산하면 된다.

가능한 모양의 끝 부분을 y, x좌표의 시작 기준점으로 정해서 범위를 벗어나는 지 체크하면서 계산하였다.
