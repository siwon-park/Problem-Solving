## [골5] 닭강정의 전설 (30972번)

https://www.acmicpc.net/problem/30972

### 문제 유형

누적합

<br>

### 어려웠던 점 / 문제의 핵심

`(i, j)`에서 `i == r1 || i == r2 || j == c1 || j == c2`이면 kij에 -1을, 아니면 1을 곱해야 한다.

먼저 2차원의 누적합 배열을 만든 다음에 `(r1, c1) ~ (r2, c2)` 구간의 누적합1을 구하고, `(r1 + 1, c1 + 1) ~ (r2 - 1, c2 - 1)` 구간의 누적합2를 구해서 `누적합2의 2배한 값에서 누적합1을 빼주면` 정답이다.
