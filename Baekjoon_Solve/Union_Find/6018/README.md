### 문제 유형

### 어려웠던 점 / 문제의 핵심

- 주어지는 M개의 간선 정보를 서로 union하여 부모를 같게 만든다.
- Q개의 쿼리에 대해서 각 정점의 부모가 같은지 확인하여 같다면 Y, 다를 경우 N을 출력한다.
