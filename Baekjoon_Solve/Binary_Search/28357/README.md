## [실2] 사탕 나눠주기 (28357번)

https://www.acmicpc.net/problem/28357

### 문제 유형

이분 탐색, 매개 변수 탐색

<br>

### 어려웠던 점 / 문제의 핵심

최대 나눠 줄 수 있는 사탕은 K개이고, K개 이하로 최대한 사탕을 나눠줄 수 있을 때 기준으로 정할 수 있는 점수 X의 최솟값을 매개 변수 탐색으로 찾으면 된다.

K와 X는 int형을 초과할 수도 있기 때문에 long형으로 선언한다. (배열도 마찬가지)

<br>
