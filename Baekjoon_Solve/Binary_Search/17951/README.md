### 문제 유형

이분 탐색, 매개 변수 탐색

### 어려웠던 점 / 문제의 핵심

그룹 수가 K이상이면 시험지 그룹의 최솟값을 늘려서 그룹의 수를 줄여야 하고, 그룹 수가 K보다 작으면, 시험지 그룹의 최솟값을 줄여서 그룹 수를 증가시켜야 한다.
