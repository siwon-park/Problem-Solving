## [실1] 곱셈 (1629번)

https://www.acmicpc.net/problem/1629

### 문제 유형

수학, 분할 정복을 이용한 거듭제곱

<br>

### 어려웠던 점 / 문제의 핵심

모듈러 연산을 활용한 메모이제이션으로 문제를 해결할 수 있다.

`ret = ((A % C) * (B % C)) % C`로 마지막에 한 번더 모듈러 연산을 해줘야 함을 유의.
