https://www.acmicpc.net/problem/16719

### 어려웠던 점 / 문제의 핵심

현재 구간에서 아직 방문하지 않았으면서 가장 작은 문자열을 찾는다.

그 후 방문한 문자열에 대해서만 버퍼에 담는다.

오름차순으로 출력해야 하므로 오른쪽 구간을 우선 탐색한 뒤에 왼쪽 구간을 탐색한다.
