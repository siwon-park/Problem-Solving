### 어려웠던 점 / 문제의 핵심

비트 연산을 통해서 백트랙킹으로 풀 수 있었는데, 어차피 1의 개수를 매번 세어야 할 것 같아서

일반적인 백트랙킹으로 풀었다.

백트랙킹으로 조합을 구성하면서, 연주할 수 있는 곡의 수가 최대일 경우에 필요한 최소 기타 수를 구하면 된다.
