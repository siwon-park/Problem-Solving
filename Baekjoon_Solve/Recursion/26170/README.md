## [실3] 사과 빨리 먹기 (26170번)

https://www.acmicpc.net/problem/26170

### 문제 유형

그래프 이론, 그래프 탐색, DFS, 백트랙킹

<br>

### 어려웠던 점 / 문제의 핵심

5 X 5 격자판을 깊이 우선 탐색으로 백트랙킹 탐색하여 사과를 3개 먹었을 때 이동한 최단 거리를 구하면 되는 문제이다.

사과를 3개 먹을 수 없을 때는 -1을 출력하면 된다.
