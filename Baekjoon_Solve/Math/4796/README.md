## [브1] 캠핑 (4796번)

https://www.acmicpc.net/problem/4796

### 문제 유형

수학, 그리디

<br>

### 어려웠던 점 / 문제의 핵심

총 휴가일 수 `V`를 연속하는 `P`일로 나눈 몫을 사용 가능한 기간인 `L`과 곱한 다음에

-  L >= V % P이면 V % P를 더한다.
- L < V % P이면 L을 더한다.

<br>
