https://www.acmicpc.net/problem/5347

### 문제 유형

수학, 정수론, 유클리드 호제법

<br>

### 어려웠던 점 / 문제의 핵심

#### gcd

b가 0보다 큰 동안 a, b = b, a % b를 계속해서 계산하여 최대 공약수를 구함

```
while (b > 0) {
	a, b = b, a % b
}
return a
```

#### lcm

최소 공배수 = 두 수의 곱 나누기 최대 공약수

```
return a * b / gcd(a, b)
```
