## [실3] 눈덩이 굴리기 (21735번)

https://www.acmicpc.net/problem/21735

### 문제 유형

브루트포스, 백트랙킹

<br>

### 어려웠던 점 / 문제의 핵심

시작 위치가 0이고, 눈덩이의 초깃값은 1이다.

백트랙킹을 통해 매번 눈덩이의 최댓값을 갱신해준다.

최적화를 위해 종료 조건을 설정하다 보니 잘못 설정하여 예외 케이스가 생겼다.
