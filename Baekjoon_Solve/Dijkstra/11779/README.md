## [골3] 최소비용 구하기 2 (11779번)

https://www.acmicpc.net/problem/11779

### 문제 유형

그래프 이론, 다익스트라

<br>

### 어려웠던 점 / 문제의 핵심

다익스트라 최단 경로 알고리즘을 통해 최단 경로를 찾으면서,

현재 위치에서 다음 위치로 이동하게 될 때, 배열에 `path[다음위치] = 현재위치`로 기록해둔다.

그 후 도착 지점에서 출발해서 베열을 역으로 탐색하여 최단 경로를 찾는다.
