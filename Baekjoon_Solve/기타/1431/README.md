### 어려웠던 점 / 문제의 핵심

문제에 조건에 맞게 정렬 조건 대로 구현하여 정렬하면 된다.

`String`의 정렬 로직에 대해서 잘 몰라서 검색해서 찾았는데,

Java에서 문자열 간 대소비교 연산은 `compareTo` 메서드를 활용해서 비교한다.

`o1.compareTo(o2)`라고 했을 때,

- o2가 사전 순으로 더 빠르면 양수를 반환한다.
- o1이 사전 순으로 더 빠르면 음수를 반환한다.
- 같으면 0을 반환한다.
